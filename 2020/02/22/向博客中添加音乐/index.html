向hexo博客中添加音乐~

2020-02-22

技术向 博客管理

1.安装aplayer插件

2.开启meting功能

aplayer:
 meting: true

MetingJS 是基于Meting API 的 APlayer 衍生播放器，引入 MetingJS 后，播放器将支持对于 QQ音乐、网易云音乐、虾米、酷狗、百度等平台的音乐播放。

3.插入歌曲或歌单

  • 插入歌曲
    {% meting "歌曲ID" "音乐平台" "song" "theme:#555" "mutex:true" "listmaxheight:340px" "preload:auto" %}

其中歌曲ID替换为网易云或其他平台播放网址中的id，比如上面的惠化洞ID为424474255,平台为netease, tencent, kugou, xiami, baidu

  • 插入歌单
    {% meting "歌单id" "音乐平台" "playlist" "theme:#555" "mutex:true" "listmaxheight:340px" "preload:auto" %}

其中歌单ID数字替换为网易云或其他平台播放网址中的id

  • 其他用法
    参考MetingJS主页：https://github.com/metowolf/MetingJS
    注意一些默认为false的功能，比如autoplay、fixed，开启时不是使用”autoplay:true”，而是直接加入”autoplay”就好了

4.获取网易云封面图片

直接用浏览的页面审查功能，找到歌曲封面比如我们这首歌的图片地址为https://p2.music.126.net/FEeBHaoHB72ydF-gRWABGw==/3444769937959625.jpg?param=34y34，去掉后面的param就为高清图的地址了
